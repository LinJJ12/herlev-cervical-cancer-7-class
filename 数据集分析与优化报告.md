## 二、优化策略

### 2.1 损失函数优化

#### 方案A: 类别加权交叉熵损失

根据分析结果，建议使用以下类别权重:

| 类别 | 权重 |
|------|------|
| 浅表鳞状上皮 | 2.66 |
| 中度鳞状上皮 | 2.81 |
| 柱状上皮 | 2.01 |
| 轻度发育不良 | 1.08 |
| 中度发育不良 | 1.35 |
| 重度发育不良 | 1.00 |
| 原位癌 | 1.31 |

#### 方案B: Focal Loss（推荐）

**原因**:
1. 类别不平衡问题明显
2. 可能存在难易样本差异

**建议参数**:
- `gamma=2.0` （控制难易样本的权重调整强度）
- `alpha=类别权重` （控制各类别的权重）

### 2.2 采样策略优化

#### WeightedRandomSampler 采样权重（反比例）:

| 类别 | 采样权重 |
|------|----------|
| 浅表鳞状上皮 | 0.0068 |
| 中度鳞状上皮 | 0.0071 |
| 柱状上皮 | 0.0051 |
| 轻度发育不良 | 0.0027 |
| 中度发育不良 | 0.0034 |
| 重度发育不良 | 0.0025 |
| 原位癌 | 0.0033 |

**特殊处理**: 对重度发育不良给予额外的权重倍数 (1.5x)

### 2.3 数据增强优化

对所有类别使用增强，对样本较少的类别使用更强的增强:

1. 随机裁剪 (RandomCrop)
2. 水平/垂直翻转
3. 旋转 (±45°)
4. 仿射变换
5. 颜色抖动
6. 透视变换

### 2.4 训练策略优化

1. **分层抽样: 确保训练/验证/测试集分布一致
2. **重点关注 F1-score 和 Recall，而非仅看准确率
3. **对少数类类别（如重度发育不良）单独评估
4. **使用混淆矩阵分析误分类模式
