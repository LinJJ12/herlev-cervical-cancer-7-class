# 宫颈涂片细胞分类 - 卷积神经网络实训报告

## 一、任务说明

### 1.1 背景
宫颈涂片检查是宫颈癌筛查的重要手段之一。本实训任务基于宫颈涂片基准数据库，使用卷积神经网络（CNN）对宫颈细胞图像进行分类，旨在推动宫颈癌筛查技术的自动化和智能化发展。

### 1.2 数据集介绍
数据集包含917张宫颈涂片细胞图像，由细胞学技术人员和医生精心标注，分为7个类别：

| 类别编号 | 类别名称（英文） | 类别名称（中文） | 性质 |
|---------|----------------|----------------|------|
| 1 | normal_superficiel | 浅表鳞状上皮 | 正常 |
| 2 | normal_intermediate | 中度鳞状上皮 | 正常 |
| 3 | normal_columnar | 柱状上皮 | 正常 |
| 4 | light_dysplastic | 轻度鳞状非角化性发育不良 | 异常 |
| 5 | moderate_dysplastic | 中度鳞状非角化性发育不良 | 异常 |
| 6 | severe_dysplastic | 重度鳞状非角化性发育不良 | 异常 |
| 7 | carcinoma_in_situ | 中度鳞状细胞原位癌 | 异常 |

其中，类别1-3为正常细胞，类别4-7为异常细胞。

## 二、方法

### 2.1 数据预处理

#### 2.1.1 数据划分
- 训练集：70%
- 验证集：15%
- 测试集：15%
- 采用分层抽样确保各类别比例在各数据集中保持一致

#### 2.1.2 数据增强（仅用于训练集）
- 随机水平翻转
- 随机旋转（±15度）
- 图像尺寸统一调整为 224×224

#### 2.1.3 标准化
使用 ImageNet 标准参数进行归一化：
- Mean: [0.485, 0.456, 0.406]
- Std: [0.229, 0.224, 0.225]

### 2.2 模型架构

本实验构建了一个4层卷积神经网络，具体结构如下：

```
CervicalCNN(
  (features): Sequential(
    (0): Conv2d(3, 32, kernel_size=3, padding=1)
    (1): BatchNorm2d(32)
    (2): ReLU(inplace=True)
    (3): MaxPool2d(kernel_size=2, stride=2)
    (4): Conv2d(32, 64, kernel_size=3, padding=1)
    (5): BatchNorm2d(64)
    (6): ReLU(inplace=True)
    (7): MaxPool2d(kernel_size=2, stride=2)
    (8): Conv2d(64, 128, kernel_size=3, padding=1)
    (9): BatchNorm2d(128)
    (10): ReLU(inplace=True)
    (11): MaxPool2d(kernel_size=2, stride=2)
    (12): Conv2d(128, 256, kernel_size=3, padding=1)
    (13): BatchNorm2d(256)
    (14): ReLU(inplace=True)
    (15): MaxPool2d(kernel_size=2, stride=2)
  )
  (classifier): Sequential(
    (0): Dropout(p=0.5)
    (1): Linear(256*14*14, 512)
    (2): ReLU(inplace=True)
    (3): Dropout(p=0.5)
    (4): Linear(512, 7)
  )
)
```

### 2.3 训练设置

- **损失函数**：交叉熵损失（CrossEntropyLoss）
- **优化器**：Adam（学习率=0.001）
- **学习率调度**：StepLR（每10轮衰减为原来的0.1倍）
- **训练轮数**：25 epochs
- **批次大小**：32
- **设备**：CUDA（如可用）或 CPU
